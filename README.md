# diabetes-clinical-trial-analysis
Statistical analysis of a 24-week diabetes clinical trial using Python (Pandas, Numpy, Scipy, Statsmodels, Matplotlib, Seaborn). Performed T-Tests and Repeated Measures ANOVA to verify medication efficacy (p &lt; 0.001) and Chi-Square tests to evaluate safety profiles. Features automated data cleaning and Seaborn visualisations for clinical reporting.
